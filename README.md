## Setup
### Run tests:

`git clone https://github.com/jams2/deobfuscate.git`

`cd deobfuscate`

`python3 -m venv ./deobfuscate`

`source deobfuscate/bin/activate`

`pip install -r requirements.txt`

`pip install -e .`

`pytest`

`coverage run -m pytest && coverage report`
